Synrc Web Framework for Erlang
==============================

Nitrogen 2 Optimized
--------------------

* Page construction from binaries
* Do all Actions through WebSocket channel
* Work within Cowboy processes
* No JSON
* No nitrogen.js
* Compatible enought to transfer Nitrogen sites

Credits

Try Immediately
---------------

    $ rebar get-deps
    $ rebar compile
    $ ./release.sh
    $ ./nitrogen_static.sh
    $ ./release_sync.sh
    $ ./start.sh
